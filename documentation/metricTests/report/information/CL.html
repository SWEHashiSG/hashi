Class Locality (CL)

Description:

CL is computed as the relative number of dependencies that a class has in its own package .

CL is computed by dividing the number of local classes on which the measured class depends by the total number of classes on which it depends;
number of local classes is computed by subtracting NOED value from the total number of classes on which the measured class depends:

(numOfImportedClasses - NOED)/numOfImportedClasses

where numOfImportedClasses is the total number of classes the measured class depends on, and the NOED is the number of its external dependencies.
